Shit Mint Linux

The purpose of Linux Mint is to produce a modern, elegant and comfortable operating system which is both powerful and easy to use. O objetivo do Linux Mint é produzir um sistema operacional moderno, elegante e confortável, que seja poderoso e fácil de usar.

Linux Mint is one of the most popular desktop Linux distributions and used by millions of people. O Linux Mint é uma das distribuições Linux de desktop mais populares e usado por milhões de pessoas.

Some of the reasons for the success of Linux Mint are: Algumas das razões para o sucesso do Linux Mint são:

It works out of the box, with full multimedia support and is extremely easy to use. Funciona fora da caixa, com suporte multimídia completo e é extremamente fácil de usar.

It's both free of cost and open source. É livre de custos e de código aberto.

It's community-driven. Users are encouraged to send feedback to the project so that their ideas can be used to improve Linux Mint. É dirigido pela comunidade. Os usuários são encorajados a enviar feedback ao projeto para que suas ideias possam ser usadas para melhorar o Linux Mint.

Based on Debian and Ubuntu, it provides about 30,000 packages and one of the best software managers. Baseado no Debian e no Ubuntu, ele fornece cerca de 30.000 pacotes e um dos melhores gerenciadores de software.

It's safe and reliable. Thanks to a conservative approach to software updates, a unique Update Manager and the robustness of its Linux architecture, Linux Mint requires very little maintenance (no regressions, no antivirus, no anti-spyware...etc). É seguro e confiável. Graças a uma abordagem conservadora para atualizações de software, um exclusivo Gerenciador de Atualizações e a robustez de sua arquitetura Linux, o Linux Mint requer pouca manutenção (sem regressões, sem antivírus, sem anti-spyware ... etc.).
